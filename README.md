## Water-quality
### 本项目主要借鉴自csdn：https://blog.csdn.net/moringacui/article/details/127374685
###      主要使用的图像为：高分二号影像0、1、2、3共四个波段的数组数据
###      通过波段运算操作，运算了汉江流域的NDWI、TP、TN、DO等水质参数
###      再使用arcgis、envi等软件进行后期处理，属于无人机和遥感技术结合的河湖水质监测技术的软件制作
